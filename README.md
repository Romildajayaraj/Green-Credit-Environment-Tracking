# Green Credit Tracking Environment System

## 📌 Overview

A MERN stack web application that encourages people to perform eco-friendly activities like tree planting, waste management, and public cleaning. Users upload activities and earn **Green Credit Scores**.

---

##  Features

* User registration & login
* Upload environmental activities with location
* Green credit score & leaderboard
* Complaint system for reporting issues
* Chat feature for user interaction

---

##  Smart Enhancements

* **Automated Scoring Algorithm**
* **Recommendation System** (suggests eco-friendly actions)
* **Priority-based Complaint Handling**

---

##  Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB

---

##  Setup

```bash
git clone https://github.com/your-username/green-credit-system.git
cd green-credit-system
npm install
npm start
```

---

##  Purpose

Promotes environmental awareness using **scoring, and smart algorithms**, with potential integration into real-world reward systems.
